基于FastAdmin的小程序api后台开发（自用）。


## **主要特性**

> 后台安装了部分FastAdmin常用插件

> 删除FastAdmin框架多余的东西

> api模块进行了路由配置 异常抛出 miss路由 微信统一支付 小程序配置等整合

> api模块服务 模型 控制器 验证器分层

> 优化halt()调试方法

## **安装使用**

>git clone https://github.com/lzqqdy/fast_api.git

>cd fast_api

>composer install

>导入base.sql

>配置虚拟站点
